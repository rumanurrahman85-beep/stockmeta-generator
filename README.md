# StockMeta Generator

## WHAT YOU DO (3 Steps)

### Step 1: Upload to GitHub
1. Go to github.com → Create new repository → Name it `stockmeta-generator`
2. Upload ALL files from this folder (index.html, css/, js/)
3. Go to Settings → Pages → Source: Deploy from a branch → main → Save
4. Wait 2 minutes. Your site is live at:
   `https://YOURUSERNAME.github.io/stockmeta-generator`

### Step 2: (Optional) Get Free API Key for Better AI
- Go to https://aistudio.google.com/app/apikey
- Sign in with Google → Click "Create API Key"
- Copy the key. Paste it in the website.
- Without key: works with local smart fallback (still good)
- With key: AI generates human-like SEO metadata (better)

### Step 3: Use It
1. Open your website
2. Drag & drop your files (or click to browse)
3. Wait for processing (green bar fills up)
4. Click the **Download All** button
5. You get 2 files: `stockmeta-metadata.csv` and `stockmeta-metadata.txt`
6. Copy-paste metadata into Adobe Stock, Shutterstock, etc.

## That's It. No Build. No Install. No Server.
